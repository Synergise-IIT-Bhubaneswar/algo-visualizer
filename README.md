# Algo-Visualizer

- An interactive tool to learn graph algorithms easily
- Built with React and MaterialUI

## Current Features

**[✔]** Add, move and delete nodes

**[✔]** Directed/Undirected edges

**[✔]** Unweighted/Weighted edges

**[✔]** Visited nodes/edges animations
